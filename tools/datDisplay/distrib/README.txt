Tool to display DAT (some) contents
===================================

Prerequisites
------------
It needs a Java Runtime Environnement.
Tested with Java 8.

Configure the file go.bat
-------------------------
1) Set the path of the Lotro client
Set the LOTRO_CLIENT_PATH variable to the path of the DAT files.

2) Parameters
--id=ID
ID is the properties file to use (1879...).

--language=LANGUAGE
where LANGUAGE is en,fr or de

Start
------
From the Windows explorer, start go.bat
